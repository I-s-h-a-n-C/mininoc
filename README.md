# Mini Network Operating Center (wifi health checker)
Arduino Code for ESP32C3 Supermini+1.8 inch IPS TFT Display (ST7735 Driver)

Power Connections
3V3                   →    FM
GND                   →    GND FF
VCC                   →    FM
LED                   →    FM

SPI Communication Connections
Pin 3                 →    SCL (SCLK) FF
Pin 4                 →    SDA (MOSI) FF
Pin 5                 →    DC FF
Pin 6                 →    RST FF
Pin 7                 →    CS FF
